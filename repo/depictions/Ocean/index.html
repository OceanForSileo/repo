Ocean For Sileo

With Ocean you can customize your experience in the new package manager for jailbroken devices, Sileo. Sileo is currently in beta testing right now but with Ocean you can unlock extra features not available to the public and some that aren't included at all (as far as the public knows). 

Use Ocean to:

    - Change the color scheme
    - Export your tweak lists
    - Auto-Respring after installation
    - Add unsupported repos*
